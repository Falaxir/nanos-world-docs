# üè† StaticMesh

A StaticMesh entity represents a Mesh which can be spawned in the world, can‚Äôt move and is more optimized for using in decorating the world.

import { Structs, BasicType, AuthorityType, Classes, Enums, AssetPath } from '@site/docs/components/_nanos.mdx';

![](/img/docs/static-mesh.jpg)

Static Meshes are like <Classes.Prop />s, but with fewer interaction options. Static Meshes are aimed to offer better performance on spawning Static "objects" in the world than Props.

:::info

üü¶üüß **Authority:** This class can be spawned on both [Client](./core-concepts/scripting/authority-concepts.mdx) and [Server](./core-concepts/scripting/authority-concepts.mdx) (if you spawn it on client, it won't be synchronized with other players).

‚è¨ **Inheritance:** This class inherits from <Classes.Actor /> and <Classes.Paintable />.

:::


## Usage

```lua
local static_mesh = StaticMesh(
  Vector(-900, 185, 215),
  Rotator(0, 90, 90),
  "nanos-world::SM_Crate_07"
)
```
