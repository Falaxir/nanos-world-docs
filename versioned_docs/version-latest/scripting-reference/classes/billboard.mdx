## 🎒 Examples

```lua title=Client/Index.lua
local my_billboard = Billboard(
    Vector(200, 200, 200), -- location
    "nanos-world::M_NanosMasked", -- material
    Vector2D(32, 32), -- size
    true
)

my_billboard:SetMaterialTextureParameter("Texture", "package://my_package/Client/image.jpg")
```
