You can use some Unreal Engine plugins when creating an asset pack for Nanos World. You can learn more about UE Plugins [here](https://dev.epicgames.com/documentation/en-us/unreal-engine/plugins-in-unreal-engine).


## List of whitelisted UE Plugins

- [Pose Search](https://dev.epicgames.com/documentation/en-us/unreal-engine/BlueprintAPI/Animation/PoseSearch)
- [Animation Warping](https://fab.com/s/e9829de17f28)
- [Motion Warping](https://dev.epicgames.com/documentation/en-us/unreal-engine/motion-warping-in-unreal-engine)
- [Chooser](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Plugins/Chooser)
- [MotionTrajectory](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Plugins/MotionTrajectory)
- [Ultra Dynamic Sky](https://fab.com/s/77a632ad8b62)
- [Chaos Vehicles](https://dev.epicgames.com/community/learning/tutorials/ZeKz/unreal-engine-setting-up-the-chaos-vehicles-plugin)
- [PCG (Procedural Generation)](https://dev.epicgames.com/community/learning/tutorials/j4xJ/unreal-engine-introduction-to-procedural-generation-plugin-in-ue5-2)
- [HDRI Backdrops](https://dev.epicgames.com/documentation/en-us/unreal-engine/hdri-backdrop)
- [Nanite](https://dev.epicgames.com/documentation/en-us/unreal-engine/nanite-virtualized-geometry-in-unreal-engine)
